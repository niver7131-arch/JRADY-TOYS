# 🎪 JRADY TOYS - Tienda en Tiempo Real con Firebase

## 📖 Descripción
Tienda online de juguetes con actualización en tiempo real usando Firebase Firestore.

## 🚀 Características
- ✅ Productos en tiempo real (Firebase Firestore)
- ✅ Panel de administración (Agregar/Eliminar productos)
- ✅ Sistema de compras con comprobante
- ✅ WhatsApp integrado para pagos
- ✅ Filtros por categoría y precio
- ✅ Búsqueda de productos
- ✅ Diseño responsive
